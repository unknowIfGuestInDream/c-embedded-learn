# 示例02：按键输入检测

## 学习目标

- 了解GPIO输入模式的配置
- 学习按键消抖（Debounce）技术
- 理解上拉/下拉电阻的作用

## 原理说明

按键是最基础的输入设备。当按下按键时，GPIO引脚检测到电平变化。由于机械按键的抖动特性，需要进行消抖处理。

### 硬件连接

```
VCC ----[上拉电阻]----+---- MCU GPIO Pin
                      |
                   [按键]
                      |
                     GND
```

- 按键未按下时，GPIO读取到高电平
- 按键按下时，GPIO读取到低电平

### 按键消抖

机械按键在按下和释放时会产生抖动，导致短时间内电平多次变化。
常用的消抖方法：
1. 软件延时消抖
2. 状态机消抖
3. 硬件RC滤波

## 代码说明

### main.c

主程序文件，检测按键状态并做出响应。

### button.h / button.c

按键控制模块，封装了按键初始化和读取功能，包含消抖处理。

## 编译运行

```bash
# 编译
make

# 运行（模拟器模式，按Enter模拟按键）
./main

# 清理
make clean
```
