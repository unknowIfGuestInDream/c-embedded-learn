# 示例01：LED闪烁

## 学习目标

- 了解GPIO（通用输入输出）的基本概念
- 学习如何控制GPIO输出高低电平
- 理解延时函数的实现原理

## 原理说明

LED（发光二极管）是嵌入式开发中最基础的输出设备。通过控制GPIO引脚输出高电平或低电平，可以控制LED的亮灭。

### 硬件连接

```
MCU GPIO Pin ----[电阻]---- LED ---- GND
```

- 当GPIO输出高电平时，LED点亮
- 当GPIO输出低电平时，LED熄灭

## 代码说明

### main.c

主程序文件，实现LED闪烁的主循环逻辑。

### led.h / led.c

LED控制模块，封装了LED初始化、开启、关闭和切换等操作。

## 编译运行

```bash
# 编译
make

# 运行（模拟器模式）
./main

# 清理
make clean
```
