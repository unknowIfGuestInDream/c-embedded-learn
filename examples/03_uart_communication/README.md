# 示例03：串口通信

## 学习目标

- 了解UART（通用异步收发传输器）的基本原理
- 学习串口通信的参数配置（波特率、数据位、停止位等）
- 掌握串口数据发送和接收的方法

## 原理说明

UART是一种异步串行通信协议，常用于嵌入式设备与PC或其他设备之间的通信。

### UART通信参数

| 参数 | 说明 | 常用值 |
|------|------|--------|
| 波特率 | 每秒传输的位数 | 9600, 115200 |
| 数据位 | 每帧数据的位数 | 8 |
| 停止位 | 帧结束标志位数 | 1 |
| 校验位 | 奇偶校验 | None |

### 数据帧格式

```
[起始位][数据位(8)][校验位(可选)][停止位]
   0    D0-D7          P           1
```
